# ParkEase-Project


ParkEase – Parking Management System is a smart solution that provides real-time information about available parking slots and helps users find and manage parking easily.  
The system reduces traffic congestion, fuel wastage, and parking time by offering efficient slot monitoring and user guidance.  
ParkEase Parking Management System using **Spring Boot + React Frontend**.

## Tech Stack
- Spring Boot (Backend)
- React (Frontend)
- PostgreSQL
- REST APIs

## Folder Structure
- frontend/ → UI
- backend/ → Spring Boot
- docs/ → Project documents
